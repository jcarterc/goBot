# goBot — Voxel Sandbox

A Minecraft-like voxel sandbox built with Godot 4.6, playable in the browser.

## Controls

| Input | Action |
|---|---|
| Click | Capture mouse / start |
| WASD | Move |
| Space | Jump |
| Shift | Sprint |
| Left-click | Break block |
| Right-click | Place block |
| 1–7 / scroll | Select block |
| Esc | Release mouse |

## Live URL

After the first GitHub Actions run completes, the game is live at:  
**https://jcarterc.github.io/goBot/**

## Setup (one-time)

After pushing this repo:
1. Go to **Settings → Pages** in your repo
2. Under "Build and deployment" → Source → select **GitHub Actions**
3. The workflow will run automatically and deploy the game
